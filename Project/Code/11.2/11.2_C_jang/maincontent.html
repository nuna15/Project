  • 예매 가능 시간

    인터넷으로 예매가 가능한 시간은 해당 공연 상영 1시간 전까지 이며 이후 예매를 원하실 경우에는 영화관 현장에 직접 가셔서 예매를 하셔야합니다.

    - 예매 취소 및 환불 가능 시간은 해당 공영 상영 20분 전까지입니다.
